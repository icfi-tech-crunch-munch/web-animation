Follow Through & Overlapping

  • Follow Through means not everything stops at once
  • Overlapping actions involve several related objects moving together, but possibly offset

Achieved through either cubic-bezier or by keyframes that move the opposite direction for a bit and then go to final destination

@keyframes followthrough {
  0% {
    transform: transform: translateX(0);;
  }
  95% {
    transform: transform: translateX(105%);
  }
  100% {
    transform: transform: translateX(100%);
  }
}
