Anticipation

  • Objects don't move without some initial movement

Similarily to follow through, achieved through either cubic-bezier or by keyframes that overshoot and then snap back

@keyframes anticipation {
  0% {
    transform: transform: translateX(0);;
  }
  5% {
    transform: transform: translateX(-5%);
  }
  100% {
    transform: transform: translateX(100%);
  }
}
